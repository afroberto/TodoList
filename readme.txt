Todo List App con Node.js

Sigue los pasos para ejecutar la aplicacion

1.- Clonar el repositorio desde el siguiente enlace
2.- Abrir una consola CMD y ve a la ruta del proyecto descargado Ejemplo: cd C:\Users\Usuario1\Desktop\CarpetaProyecto
3.- Instala las dependcias del proyecto con el siguiente comando comando: npm install
4.- Ejecuta la aplicacion con el siguiente comando: node index.js
5.- Por ultimo en tu navegador ve a esta direccion http://localhost:8080

Para detener la aplicacion presiona ctrl+c
