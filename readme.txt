Todo List App con Node.js

Sigue los pasos para ejecutar la aplicacion

1.- Descargar GitBash o instalar Git en tu equipo, para la descarga de GitBash lo puedes hacer desde el siguiente enlace: https://www.git-scm.com/downloads
1.- Abrir GitBash o CMD en la ruta donde se clonara el proyecto, Clonar el repositorio desde el siguiente enlace: git clone https://github.com/afroberto/TodoList.git
2.- Posicionate dentro de la carpeta del proyecto descargado Ejemplo: cd C:\Users\Usuario1\Desktop\CarpetaProyecto
3.- Instala las dependcias del proyecto con el siguiente comando: npm install
4.- Ejecuta la aplicacion con el siguiente comando: node index.js
5.- Por ultimo en tu navegador ve a esta direccion http://localhost:8080

Para detener la aplicacion presiona ctrl+c
